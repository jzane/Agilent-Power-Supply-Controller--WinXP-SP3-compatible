With the power supply connected to a usb port, open the program, select the port the device is connected to, and click 'connect'.
Choose from one of the tab windows, and enter values for fields such as 'current limit', 'voltage', or 'rise time'.
Click 'SET!' to send command to the device.
The program handles closing of the port automatically when exited.

targeted Win XP sp 3 (32 bit OS)
